⬆️ Nhập dữ liệu từ Excel

Tải lên file .xlsx đã chỉnh sửa để cập nhật hoặc thêm mới dữ liệu vào hệ thống. Dữ liệu trùng tên sẽ được cập nhật, dữ liệu mới sẽ được thêm vào.

⚠️ Lưu ý: Chỉ import file từ chức năng "Xuất tất cả" hoặc "Tải Template" ở trên. Không thay đổi tên cột hoặc cấu trúc file.
